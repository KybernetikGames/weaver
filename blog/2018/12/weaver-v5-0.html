Weaver v5.0 - Get rid of your magic strings

Published
Monday, 3 December 2018
Category
Release
Author
Kailas

Have you ever wanted to:

  • Use [SerializeField] on a static field or property to get an asset reference?
  • Get a strongly typed list of assets in a particular folder?
  • Stop using magic strings to access animations, layers, navigation areas, scenes, and shaders?
  • Generate procedural assets in the Unity Editor which can then be used like regular assets?

Well, I just released a massive update to my Weaver plugin which lets you do all those things. Please check it out and let me know if you have any questions, feedback, feature requests, etc.

If you are upgrading from an earlier version of Weaver, the Change Log contains an Upgrade Guide to help you.
